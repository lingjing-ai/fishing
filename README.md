# HTML5 Canvas 捕鱼游戏

一个基于HTML5 Canvas技术开发的简单而有趣的捕鱼游戏。玩家通过控制炮台发射子弹，击中移动的目标来获取积分。

## 功能特点

- 流畅的Canvas动画效果
- 可旋转的炮台系统
- 物理碰撞检测
- 动态积分系统
- 可调节的子弹速度
- 随机生成的移动目标
- 边界反弹的子弹效果

## 技术实现

- 使用HTML5 Canvas进行游戏渲染
- 原生JavaScript实现游戏逻辑
- 面向对象的代码结构
- requestAnimationFrame实现流畅动画
- 基于时间的游戏循环系统

## 游戏玩法

1. 控制方式：
   - ← →方向键：控制炮台旋转
   - ↑ ↓方向键：调整子弹速度
   - 空格键或鼠标点击：发射子弹

2. 计分规则：
   - 初始积分：100分
   - 发射子弹：-1分
   - 击中目标：+2分

3. 游戏技巧：
   - 合理调整子弹速度以适应不同距离的目标
   - 注意观察目标移动轨迹，提前预判射击位置
   - 保持足够的积分以持续游戏

## 快速开始
